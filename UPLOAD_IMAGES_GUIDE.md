# How to Upload Your Real Project Images

## Quick Start Guide

Your website is now set up to display your actual project photos! Here's how to upload them:

### Step 1: Open Object Storage
1. In Replit, look for the **"Object Storage"** pane in your workspace
2. If you don't see it, click the tools icon (🔧) and add "Object Storage"

### Step 2: Create Folders
1. In Object Storage, navigate to the `public` folder
2. Create a new folder called `case-studies`

### Step 3: Upload Your Images
For each project, upload an image with the following naming format:
- The filename should match your client name (lowercase, with hyphens)
- Use .jpg format for best compatibility

**Examples:**
- Microsoft (NRF) → `microsoft-nrf.jpg`
- Netflix → `netflix.jpg`
- MrBeast → `mrbeast.jpg`
- Mac Cosmetics → `mac-cosmetics.jpg`
- Buffalo Wild Wings → `buffalo-wild-wings.jpg`
- Williams Racing → `williams-racing.jpg`

### Step 4: Upload Process
1. Click the "Upload" button in Object Storage
2. Navigate to: `public/case-studies/`
3. Select your image files
4. Upload them with the correct names

### Step 5: See Your Images Live!
Once uploaded, your images will automatically appear on the website. The system will:
- First try to load your uploaded image
- If not found, it will show a professional stock photo as fallback

## Tips:
- Images should be optimized for web (1200x800px recommended)
- Use .jpg format for photos
- Keep file sizes under 1MB for fast loading
- Your actual photos will make your portfolio much more authentic!

## Need Help?
If images aren't showing:
1. Check the filename matches exactly (case-sensitive)
2. Ensure images are in `public/case-studies/` folder
3. Refresh the page after uploading

Your Google Drive folders are still linked for full project access - visitors can click "View Real Project Assets" to see complete project documentation!
